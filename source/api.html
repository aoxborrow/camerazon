General

Products

GET/api/products.json - list all products

GET/api/products/147064802.json - single product details with variants

GET/api/products/147064802/variants.json - list just the variants for a product

POST/api/products.json - sync products with Shopify and return updated list


Variants

GET/api/variants/335858738.json - single variant details

PUT/api/variants/335858738.json - update a variant, e.g. {"inventory_quantity":"23","updated_at":"2013-07-12T03:19:10-07:00"}
